# Documentation Hub

This folder captures planning, roadmaps, and launch checklists for the Crypto Tarot universe.

## Files

- `PROJECT-PLAN.md` – Detailed plan, completed features, and phased TODOs
- `ROADMAP.md` – High-level overview of tracks and where assets live
- `TODO.md` – Current finalization checklist across all workstreams

## Guidelines

- Add new reference docs here (e.g., launch checklist, localization plan, QA notes)
- Cross-link major updates between `PROJECT-PLAN.md` and `ROADMAP.md`
- Keep dated decisions or meeting notes in this folder for easy lookup

_"Clear documentation keeps the oracle aligned."_ ✨
