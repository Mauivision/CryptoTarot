## 🎯 Common Tasks

### I want to...

#### Start Developing

1. Read [UTOPIA-GUIDE.md](../UTOPIA-GUIDE.md)
2. Check [QUICK-START.md](../docs/guides/QUICK-START.md)
3. Run `npm run serve`
4. Open `http://localhost:8000`

#### Update Card Data

1. Edit `data/card-meanings.js`
2. Run `npm run format`
3. Test in browser

#### Add New Images

1. Place images in `tools/CryptoTarot1-78/`
2. Run `npm run update:cardmap`
3. Images will be mapped automatically

#### Generate Book

1. Run `npm run generate:book`
2. Output in `print/book/`

#### Add Game Mechanics

1. Run `npm run game:add-mechanics`
2. Test with `npm run game:test`
3. Check balance with `npm run game:balance`

#### Export to PDF

1. Run `npm run export:pdf`
2. Output in `print/book/`
