## 🎴 Card Dielines

### Standard Tarot Card

- **Size:** 2.75" × 4.75" (70mm × 120mm)
- **Bleed:** 0.125" (3mm) on all sides
- **Safe Zone:** 0.25" (6mm) from edges
- **Corner Radius:** 0.125" (3mm)
- **File:** `card-dieline.pdf` (to be created)

### Dieline Specifications

```
Total Size: 3" × 5" (with bleed)
Safe Area: 2.25" × 4.25" (content area)
Bleed Area: 0.125" on all sides
Corner Radius: 0.125"
```

## 🎨 Design Guidelines

### Color Specifications

- **Print Mode:** CMYK
- **Resolution:** 300 DPI minimum
- **Color Profile:** ISO Coated v2 (ECI)

### Typography

- **Body Text:** Crimson Text, 10-12pt
- **Headers:** Helvetica Neue, 14-18pt
- **Code:** Fira Code, 9-10pt (if needed)

### Image Requirements

- **Format:** PDF/X-1a or PDF/X-4
- **Resolution:** 300 DPI
- **Color Mode:** CMYK
- **Bleed:** Include 0.125" bleed on all sides

## 🛠️ Tools & Software

### Recommended Software

- **Adobe Illustrator** - Vector dielines

- **Source:** AI (Illustrator) or INDD (InDesign)

- **Backup:** SVG or EPS
